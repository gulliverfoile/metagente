📘 README · Fábrica Abstracta de Agentes

Filosofía: Demostrar que un "agente IA" comercial a menudo es solo un motor de reglas + plantilla. Esta herramienta permite crear recomendadores personalizados sin backend y 100% configurables.

Características:

  • ✅ Motor de reglas declarativo (JSON).
  • ✅ Adaptadores de datos: JSON local, API externa (con mapeo), búsqueda Google.
  • ✅ Memoria real (historial y no repetición).
  • ✅ Exportación a HTML autónomo (sin panel de configuración).
  • ✅ Validación con JSON Schema.

Uso: Configura las dimensiones, aplica cambios, y prueba el agente en vivo. Exporta el HTML para compartir.

Nota: Esta versión no incluye generación por IA (se eliminó para mantener la honestidad del proyecto).
