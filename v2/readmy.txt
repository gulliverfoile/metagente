**Demuestra que el 80% de los "agentes IA" modernos son solo un JSON de configuración, un `filter()` y una plantilla de texto. Sin magia, sin backend, sin suscripción.**

Crea tu propio agente (recomendador de recetas, generador de excusas, asesor de viajes, analista de ventas...) en segundos respondiendo preguntas abstractas.

---

## 📖 Filosofía

> *"Un agente es un prompt bien ejecutado, que venden que es la leche pero no dejan de ser loops y eso se puede hacer con prompts bien ejecutados, que son humo."*

Este proyecto nació de una conversación sobre la verdadera naturaleza de los "Agentes IA" que inundan el marketing tecnológico. Queríamos demostrar que con un simple formulario de configuración, un motor de reglas declarativas y una plantilla, se puede replicar la funcionalidad básica de muchos agentes comerciales.

**No es magia, es abstracción.**

---

## ✨ Características

- **100% configurable**: Todas las dimensiones (objetivo, perfil, restricciones, etc.) son editables en tiempo real.
- **Motor de reglas declarativas**: Define filtros con JSON (`{ "campo": "kcal", "operador": "<", "valor": 400 }`).
- **Adaptadores de datos intercambiables**: JSON local, API externa (con mapeo de campos), o búsqueda en Google (parche low-cost).
- **Memoria real**: Historial en `localStorage` y opción de no repetir las últimas N recomendaciones.
- **Validación robusta**: JSON Schema + Ajv garantiza que la configuración sea correcta.
- **Exportación a HTML autónomo**: Genera un archivo `.html` con el agente completo listo para usar (sin el panel de configuración).
- **Traductor IA opcional**: Describe el agente en lenguaje natural y una IA (OpenAI) genera la configuración por ti (requiere tu propia API Key).
- **Sin backend, sin costes**: Funciona 100% en el navegador. Solo necesitas abrir el archivo.

## 🚀 Cómo usarlo

### Opción 1: Probar online (GitHub Pages)
*Próximamente: enlace a la demo desplegada.*

### Opción 2: Ejecutar en local
1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/fabrica-abstracta-agentes.git
Abre el archivo index.html en tu navegador favorito.

¡A crear agentes!

Modo Creador (parte superior)
Modifica el nombre, propósito y fuente de datos.

Cambia las preguntas, opciones y reglas declarativas de cada dimensión.

Pulsa "Aplicar configuración" para ver los cambios en la vista previa.

Modo Usuario (parte inferior)
Rellena el formulario generado con tus preferencias.

Pulsa "Ejecutar Agente".

Observa el resultado y el historial de recomendaciones.

Ejemplos rápidos
Recetas: Objetivo "Perder peso", restricción "Sin lactosa", tiempo 15 min.

Excusas: Modifica el JSON para que filtre por "credibilidad" y "tiempo de redacción".

Búsqueda Google: Configura cualquier dimensión y el agente abrirá una pestaña con los resultados.

Exportar agente
Una vez configurado a tu gusto, pulsa "Exportar agente como HTML completo".

Obtendrás un archivo autocontenido que solo muestra la interfaz de usuario, listo para compartir.

Usar el Traductor IA (opcional)
Introduce tu API Key de OpenAI (no se almacena en ningún servidor).

Describe el agente que quieres (ej: "Quiero un agente que recomiende películas según mi estado de ánimo y la duración máxima").

La IA rellenará el formulario automáticamente (requiere conexión a internet).
